Web Application with Web Services and JavaScript UI

This GitHub page is the home of the source code for a demo application that I wrote a while back for a potential employer. While I ultimately decided not to pursue the position, I think that the application is still a good demonstration of my ASP.Net, C# and JavaScript skills. Of course if I were doing this application again today, I would do it MVC/WebAPI, and maybe use Knockout or Angular on the client side. But this is a snapshot of the way I was doing things circa 2011.

Use the links at the top of the page to download the source code or clone the repo using Git.

A live working demo can be found here.

Application Highlights Include:
  • ASP.Net Server-Side Controls
  • HTML with CSS and JavaScript/jQuery
  • An ASP.Net Web Service
  • ASP.Net AJAX
  • jQuery AJAX
  • jQuery UI Scripting with jQuery Datatables
  • Entity Framework ORM for Database Access and CRUD
  • The Microsoft "Pubs" SQL Server Database
  • User Input Validation
  • Login/Logoff Capabilities
