Come One, Come All to the Showcase of ASP.Net Web Application Building Skills!

This GitHub repository is the home of the source code for a demo application that I wrote a while back for a potential employer. While I ultimately decided not to pursue the position, I think that the application is still a good demonstration of my ASP.Net and C# skills.

A live working demo can be found here.

Application Highlights Include:
  • ASP.Net Server-Side Controls
  • HTML Client-Side Controls
  • An ASP.Net Web Service
  • ASP.Net AJAX
  • jQuery AJAX
  • jQuery UI Scripting with jQuery Datatables
  • Entity Framework ORM for Database Access and CRUD
  • The Microsoft "Pubs" SQL Server Database
  • User Input Validation
  • Login/Logoff Capabilities
A word document containing the specific requirements given to me can be downloaded here.
